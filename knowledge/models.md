# 모델 안내 (요약·채팅용)

이 앱은 로컬 [Ollama](https://ollama.com) + Google **Gemma** 모델로 동작합니다.
아래는 어떤 모델을 쓰고, 어떤 PC에서 돌아가는지에 대한 정리입니다.

## 기본 모델 정책

- **기본 선호 = `gemma3:12b`** (요약 품질 우선).
- 선택 우선순위(앱이 자동으로 고름):
  1. `gemma3:12b` 가 설치돼 있으면 그것
  2. 없으면 설치된 **다른 12B**(예: `gemma4:12b-it-qat`)
  3. 그것도 없으면 마법사 추천 모델 → 첫 번째 설치 모델
- 상단(채팅 탭) **모델 드롭다운**에서 언제든 바꿀 수 있습니다. PDF 요약도 여기서 고른 모델을 그대로 사용합니다.

> 정식 `gemma3:12b` 를 쓰려면: 터미널에서 `ollama pull gemma3:12b` (약 8GB 다운로드).

## 요약 품질: 4B vs 12B

| 항목 | gemma3:4b | gemma3:12b |
|---|---|---|
| 요약 품질 | 보통 (빠른 초안용) | 좋음 (핵심 보존·왜곡 적음) |
| 길이 지시 준수 | 약함 (10% 요청 → 20%+ 나오기도) | 비교적 정확 |
| 한국어 표현 | 무난 | 자연스러움 |
| 속도 | 빠름 | ~3배 느림 |

- **권장**: 최종 요약은 12B, 빠른 미리보기·초안은 4B.
- 요약은 실시간이 아니므로 품질이 중요하면 12B가 유리합니다.

## 모델별 요구 사양

| 모델 | 디스크 | 권장 RAM | 권장 VRAM(GPU) | 이미지 |
|---|---|---|---|---|
| gemma3:1b | ~0.8GB | 2GB+ | 2GB+ | ✗ |
| gemma3:4b | ~3.3GB | 8GB+ | 5.8GB+ | ✓ |
| gemma3:12b | ~8GB | 16GB+ | 11GB+ | ✓ |
| gemma3:27b | ~17GB | 32GB+ | 22GB+ | ✓ |

> VRAM 은 "GPU에 전부 올려 빠르게 돌리기 위한 권장치"입니다. 부족해도 **CPU/RAM 으로 실행은 가능**하지만 느려집니다.

## 일반 PC(회사 노트북 아님)에서도 설치되나?

**됩니다.** 전용 GPU가 없어도 동작합니다. 필요한 것:

- **OS**: Windows 10/11 · macOS(Apple Silicon/Intel) · Linux
- **Python 3.10+** (설치돼 있어야 `run.bat`이 라이브러리를 자동 설치)
- **Ollama** 설치 + 실행 ([ollama.com](https://ollama.com))
- **인터넷**: 모델 최초 1회 다운로드 때만 필요 (이후 오프라인 사용 가능)
- **디스크 여유**: 위 표의 모델 크기만큼

PC 사양별 권장:

| PC 상황 | 권장 모델 | 비고 |
|---|---|---|
| 전용 GPU 없음 / RAM 8GB | `gemma3:1b` ~ `gemma3:4b` | CPU 실행. 12B는 매우 느림 |
| RAM 16GB, GPU 없음/약함 | `gemma3:4b` (요약은 12B도 가능하나 느림) | 12B는 시간 여유 있을 때 |
| GPU VRAM 8GB+ | `gemma3:4b` 쾌적, `gemma3:12b` 가능 | |
| GPU VRAM 12GB+ / RAM 16GB+ | `gemma3:12b` 쾌적 | 요약 품질 최적 |

**요점**: 일반 PC에서도 설치·실행은 되지만, **12B를 쾌적하게 쓰려면 RAM 16GB 이상(가능하면 GPU VRAM 12GB+)**을 권장합니다. 사양이 낮으면 4B로 두고 쓰면 됩니다(앱 드롭다운에서 선택).

## 모델 받기 / 확인

```bash
ollama pull gemma3:12b     # 요약 품질용 (권장)
ollama pull gemma3:4b      # 빠른 초안용
ollama list                # 설치된 모델 목록 확인
```
