# [공유] 요약집 메이커 — 수백 페이지 기본서도 통째로 (로컬 AI · Gemma3 · 인터넷 불필요)

> 원제: **LONGTEXT2SUMMARYMAKER**
> 📥 받기(GitHub): https://github.com/leedonwoo2827-ship-it/gemma-local-wizard

안녕하세요. 인터넷 연결 없이 **내 컴퓨터 안에서만 도는 AI**를 버튼 몇 번으로 설치·사용할 수 있는 사내 툴을 공유합니다.

## 한 줄 요약
클라우드(ChatGPT 등)에 올리기 부담스러운 **내부 문서·교재·자료**를, 외부로 전송하지 않고 **내 PC에서 직접 요약·질의응답**할 수 있는 데스크톱 앱입니다. (Google Gemma 모델 사용)

## 이런 분께 좋아요
- 📄 **긴 PDF·워드 문서를 빠르게 요약**하고 싶은 분 (교재/보고서/논문 등)
- 🔒 회사 자료를 **외부 AI 서비스에 올리기 꺼려지는** 분 (데이터가 PC 밖으로 안 나갑니다)
- 💬 인터넷 없이도 **AI와 대화**하고 싶은 분

## 주요 기능
1. **자동 설치 마법사** — 내 PC 사양(RAM·GPU)을 감지해 **알맞은 모델을 추천**하고, 엔진(Ollama)과 모델을 버튼으로 설치합니다. *(Docker 불필요)*
2. **AI 채팅** — 스트리밍 대화, 문서 기반 질의응답(지식 도서관), 대화 저장/검색
3. **문서 요약** — PDF·Word(docx)·txt 파일을 **원문의 10% 또는 5% 분량**으로 요약. 긴 책도 구간별로 나눠 처리하고, **결과를 시험 정리 노트처럼 계층형 개조식**(`1. → (1) → 1) → ① → ㉠ …`)으로 정리해 줍니다. 구간마다 **원문 시작 페이지(p.○○)**도 표시해 원문 대조가 쉽습니다.

## 사용 방법 (아주 간단)
1. **다운로드** — GitHub(https://github.com/leedonwoo2827-ship-it/gemma-local-wizard) → 초록색 **`Code`** 버튼 → **`Download ZIP`** → 압축 해제
2. 앱 폴더에서 **`run.bat` 더블클릭** (필요한 것들이 자동 설치됩니다)
3. 창에서 **① Ollama 설치 → ② 추천 모델 다운로드** 버튼 클릭
4. **`채팅 열기`** → 채팅하거나, 상단 **`PDF 요약`** 탭에서 문서 선택 후 요약

> 💡 처음 모델 다운로드만 인터넷이 필요하고, 이후에는 오프라인으로 사용 가능합니다.

## 권장 사양 & 어떤 모델을 쓸까?

- OS: Windows 10/11 · macOS · Linux
- 앱을 켜면 **내 PC 사양을 감지해 알맞은 모델을 자동 추천**해 줍니다. 아래는 참고표입니다.

| 내 PC RAM | 추천 Gemma 3 모델 | 비고 |
|---|---|---|
| **8GB** | **Gemma 3 · 4B** | 8GB에서 가장 무난. 일반 노트북 OK |
| 16GB 이상 | **Gemma 3 · 12B (앱 기본값)** | 요약 품질 우수. 기본으로 설정돼 있음 |
| 32GB 이상 | 27B도 가능 | 매우 무거워 느림 |
| 저사양/구형 | 1B | 가볍지만 품질은 낮음 |

> 💡 **앱 기본 모델은 12B**입니다. 다만 **RAM이 8GB라면 `3. 모델 선택`에서 `Gemma 3 (4B)`로 바꿔서** 사용하세요. (12B는 RAM 16GB 이상 권장)
> 그래픽카드(VRAM)가 넉넉하면 더 빠르게 돌지만, 없어도 CPU/RAM으로 실행됩니다. 요약은 실시간이 아니라 4B로도 충분히 쓸 만합니다.
