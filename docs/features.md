# 기능 안내

## 1) PDF 요약

- **입력**: 텍스트 복사가 가능한 PDF(또는 txt/md). 스캔 이미지 PDF 는 텍스트가 추출되지 않아 불가.
- **요약 분량**: **10%** 또는 **5%** 선택 (원문 대비 대략적인 분량).
- **긴 문서 처리**: 문서를 구간(청크)으로 나눠 각 구간을 요약한 뒤 이어 붙임 → 수백 페이지 책도 처리.
- **진행률/중지**: 구간별 진행률 표시, 중간에 중지 가능.
- **결과**: 요약 본문을 화면에서 확인 → `복사` 또는 `저장`(.md/.txt). "원문 N자 → 요약 M자 (P%)" 통계 표시.

## 2) AI 채팅

- **스트리밍 대화**: 전송 / 중지 / 재생성
- **대화 관리**: 새 대화 / 검색 / 이름변경 / 삭제, 자동 저장
- **고급 설정**: temperature, top-p, 컨텍스트 길이(num_ctx)
- **내보내기**: Markdown / 텍스트

## 모델
- 상단바에서 설치된 `gemma*` 모델을 자동 감지해 선택합니다.
- 기본값은 `gemma3:4b`. 모델이 없으면 터미널에서 `ollama pull gemma3:4b` 로 받으세요.

| 모델 | 디스크 | 권장 RAM | 권장 VRAM |
|---|---|---|---|
| gemma3:1b | ~0.8GB | 2GB+ | 2GB+ |
| gemma3:4b | ~3.3GB | 8GB+ | 5.8GB+ |
| gemma3:12b | ~8.1GB | 12GB+ | 11GB+ |
| gemma3:27b | ~17GB | 24GB+ | 22GB+ |
