# 기능 안내

## 1) 설치 마법사

- **내 컴퓨터 사양 감지**: OS / RAM / VRAM / GPU 자동 인식
- **VRAM 기준 추천 모델**: 예) VRAM 8GB → `gemma3:4b` (GPU 가속), 제한 요인(VRAM/RAM) 표시
- **Ollama 자동 설치**: 상태 표시(실행중/설치됨/없음) + 원클릭 설치
- **Gemma 모델 관리**: 설치 / 업데이트 / 삭제 / 터미널 실행, 다운로드 진행률 표시

## 2) AI 채팅 (앱 내장 · Docker 불필요)

- **스트리밍 대화**: 전송 / 중지 / 재생성
- **지식 도서관 (RAG)**: txt·md·pdf·docx 업로드 → 근거 기반 답변
  - 임베딩 모델 `nomic-embed-text` 자동 다운로드, 문서를 조각내어 벡터 검색
- **봇 관리**: 커스텀 시스템 프롬프트/페르소나 (요약 도우미, 문제집 출제자 등 기본 제공)
- **이미지 입력(멀티모달)**: `gemma3` 4B/12B/27B 에서 지원 (1B 은 텍스트 전용 — 체크박스 자동 비활성)
- **대화 관리**: 새 대화 / 검색 / 이름변경 / 삭제, 자동 저장
- **프롬프트 템플릿**: 요약·문제집·번역 등 원클릭 삽입
- **고급 설정**: temperature, top-p, 컨텍스트 길이(num_ctx)
- **내보내기**: Markdown / 텍스트

## 모델 추천 기준 (VRAM)

| VRAM | 추천 모델 | 비고 |
|---|---|---|
| < 4GB / 통합 그래픽 | `gemma3:1b` | CPU/부분 가속 |
| 4–11GB | `gemma3:4b` | 8GB 노트북 기준 균형형 |
| 12–23GB | `gemma3:12b` | |
| ≥ 24GB | `gemma3:27b` | |

- RAM 이 모델 요구치보다 부족하면 한 단계 낮은 모델을 추천하고 **"제한 요인: RAM"** 으로 표시합니다.
- macOS(Apple Silicon)는 통합 메모리를 GPU 와 공유하므로, 사용 가능한 메모리 기준으로 추천합니다.

| 모델 | 디스크 | 권장 RAM | 권장 VRAM | 이미지 |
|---|---|---|---|---|
| gemma3:1b | ~0.8GB | 2GB+ | 2GB+ | ✗ |
| gemma3:4b | ~3.3GB | 8GB+ | 5.8GB+ | ✓ |
| gemma3:12b | ~8.1GB | 12GB+ | 11GB+ | ✓ |
| gemma3:27b | ~17GB | 24GB+ | 22GB+ | ✓ |
